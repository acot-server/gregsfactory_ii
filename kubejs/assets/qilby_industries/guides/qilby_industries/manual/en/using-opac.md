# Using Open Parties & Claims
[Voir cette page en français](/fr/utiliser-opac.md)

## How to claim chunks
Open JourneyMap (with **<KeyBind id="key.journeymap.map_toggle_alt" />**) and click on the chunks you want to claim while holding **<KeyBind id="key.journeypac.claim_mode" />**.

## Set permissions within claims
*Open Parties & Claims* is highly configurable and granular in its permission system. There is a great many options that configurable via the following command:
```
/openpac player-config set <permission> <valeur>
```
To know more about what an option does, you can run
```
/openpac player-config help <permission>
```
Note however that a number of permissions cannot be changed by means of this commands, and can instead only be changed in the server’s configuration.
